Programmable Virtual Camera Access Method library
Triggering Table

On cameras where this feature is supported, the triggering table allows cycling the exposure signal on multiplexed Expose Out trigger outputs. The multiplexer will cycle the signal it is connected to in a round-robin fashion over all of the configured outputs.

The configuration is performed in two steps. The output signal to be configured can be selected via parameter PARAM_TRIGTAB_SIGNAL. The number of active output wires to be used for the selected signal can be changed by the PARAM_LAST_MUXED_SIGNAL parameter.

By default, the configured number of multiplexed outputs is set to one for all the signals. This has the same effect as if no multiplexer was connected in the signal path.
